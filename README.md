The repository aims to simulate optical propagation in turbulent media.
This is based on Jason Schmidt's book on 'Numerical Simulation of Optical Propagation'.
The book can be found on the SPIE digital library:
  https://www.spiedigitallibrary.org/ebooks/PM/Numerical-Simulation-of-Optical-Wave-Propagation-with-Examples-in-MATLAB/eISBN-9780819483270/10.1117/3.866274
